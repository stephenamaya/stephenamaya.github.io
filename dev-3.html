Minty Framework

The project was create a CSS framework & HTML boilerplate to streamline static website development. As a front end developer there are infinite frameworks, libraries, templates, and css normalizes. I decided to create my own by combining reliable elements of other examples, I call it Minty. Included is a templte website, CSS grid using a 12 column grid, and a pre-written table of contents to avoid spaghetti styling.

Challenges and Solutions

Some of the challanges involved with minty were the browser cross-compatability testing, and setting up the github pages site / downloadable poroject.


During the development process, extensive testing was done to ensure compatability specifically on Internet Explorer and Edge, these browsers are still widely used in government internet and intranet applications, and therefore CSS comprimises needed to be made in order to allow for consistent interface experiences.
