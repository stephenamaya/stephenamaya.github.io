Terror Incognita The Board Game

Terror Incognita is Project Labyrinth's first board game in development. A narrative adventure card game set in a tragic dark fantasy setting.

My Role

Graphic design and Interface for all physical and digital products. To include merchandise, box sets, card front and backs, and posters.

Challenges and Solutions

The main challenge is the evolving nature of a product that is in development. Typically, I used agile project management tools such as Git and Jira, so adjusting to a soley graphical approach was a challenge. File organization and iteration was key, researching and understanding the briefs, creating concept art, and gathering feedback was the key to speedy development and refinement.
